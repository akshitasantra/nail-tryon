# 💅 polish.exe

**polish.exe** is a virtual nail try-on web app that lets users preview nail designs on realistic hand photos before committing IRL.  
It’s built with React + Canvas and focuses on cute UI, realism, and inclusive skin tone representation.

Think Pinterest nail inspo, but you actually get to see if it eats on *your* hands.

---

## ✨ Features

- 🤎 **Real hand photo presets**  
  Three royalty-free hand images (light / medium / dark) to preview designs realistically

- 💅 **Canvas-based nail overlays**  
  Nail designs are rendered directly onto each nail using precise coordinates and rotation

- 🎨 **Preset nail designs**
  - French tips
  - Chrome
  - Aura
  - Jelly
  - Glazed donut
  - Floral
  - Solid colors (lavender, coral, nude, mint, cherry, etc.)

- 📌 **Pinterest inspo upload**  
  Upload any image and map it directly onto the nails as a texture

- 💾 **Save your look**  
  Download your final nail preview as a PNG

- 🎀 **Y2K pixel UI vibes**  
  Custom loading screen, fake window chrome, pixel fonts, pastel gradients, and sparkles

---

## 🛠️ Tech Stack

- **React**
- **HTML Canvas** for rendering nail designs
- **CSS** (custom styling, no UI libraries)
- **JavaScript** (hooks + modular architecture)
